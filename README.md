# rly-network-mobile-sdk

An SDK to make working with token economies as easy as credit cards

## Installation

```sh
npm install rly-network-mobile-sdk
```

## Usage

```js
import { getAccount } from 'rly-network-mobile-sdk';

// ...

const account = await getAccount();
```
